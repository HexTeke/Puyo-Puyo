Java remake of Puyo Puyo / Dr. Robotnik's Mean Bean Machine for the Sega Genesis

# Controls
- Arrow keys to navigate/move
- `Escape` - Back/Pause
- `Enter` - Select
- `A/S` - Rotate

# Available Content (not much)
- Preview Exercise Mode
- Options
